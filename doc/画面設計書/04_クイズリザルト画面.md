# [画面4] クイズリザルト画面 (Quiz Result)

- 目的: クイズ終了後、プレイヤーの最終結果と順位を表示する。

- 主要コンポーネント:
    - タイトル: クイズ結果
    - 最終順位リスト: プレイヤー名、アイコン、スコア、順位を表示
    - ナビゲーションボタン:
        - [設定に戻る]
        - [ゲームをリセット]
        - [ユーザーを追加する]

- レイアウト図 (アスキーアート):

```
+------------------------------------------------------+
| クイズ結果                                           |
+======================================================+
|                                                      |
|   最終順位                                           |
|   プレイヤーのスコア結果です                         |
|                                                      |
|   1. 🥇 👤田中 (10 ポイント)                         |
|   2. 🥈 🤖鈴木 ( 8 ポイント)                         |
|   3. 🥉 🦊佐藤 ( 5 ポイント)                         |
|   ...                                                |
|                                                      |
+------------------------------------------------------+
| [設定に戻る] [ゲームをリセット] [ユーザーを追加する] |
+------------------------------------------------------+
```
